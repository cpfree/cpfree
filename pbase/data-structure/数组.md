# 数组

## 一维数组

一维数组在内存中一般是连续的

## 多维数组

二维数组在内存中的结构未必是连续的

   > C++ 中的二维数组在内存中的结构是连续的
   >
   > Java 中的二维数组在内存中的结构不是连续的.
