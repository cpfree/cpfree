## 3. 以通信的方式共享内存, 而不是以共享内存来通信

```java

public class A {

   private Object b;

   // 以共享内存来通信, 不要这样调用b
   public void func1() {
      System.out.println("正在使用b: " + b);
   }

   // 以通信的方式共享内存, 需要做到这样
   public void func2(Object b) {
      System.out.println("正在使用b: " + b);
   }

   // 可以考虑和变量区分开
   public void func2(Object c) {
      System.out.println("正在使用: " + c);
   }

}

```
