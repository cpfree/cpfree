# 安装配置文件编辑规则

## 

1. 所有安装配置文件以插件名称开头
2. 安装配置写在同一个文件里面
3. 不同环境的安装配置写在不同的文件里面


安装配置相关比较全的写法以linux里面的为主.
   > 遇到window系统和linux系统安装配置相同的步骤写道linux里面, window里面直接引用linux里面的内容.
